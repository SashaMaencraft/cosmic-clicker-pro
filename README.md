# 🚀 Cosmic Clicker Pro

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://python.org)

**Multiplatform Clicker Game** - Играйте на любом устройстве! 📱💻

## 🎮 Особенности

- ✅ **Кроссплатформенность** - работает на Windows, Android и в браузере
- 💰 **Множество валют** - рубли, биткоины, акции
- 🏠 **Недвижимость** - пассивный доход каждую секунду
- 📈 **Биржа** - покупайте и продавайте акции
- 🚗 **Транспорт** - улучшайте эффективность бизнеса
- 💼 **Портфель** - отслеживайте все активы
- 📱 **Адаптивный интерфейс** - оптимизирован для всех устройств

## 🚀 Быстрый старт

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/SashaMaencraft/cosmic-clicker-pro.git
cd cosmic-clicker-pro

# Установите зависимости
pip install -r requirements.txt

# Запустите игру
python src/cosmic_clicker_crossplatform.py
